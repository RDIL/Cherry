Here is a list of features included in the client:

- ToggleSprint
- Optionally toggle the game's built-in tutorial (handy if you switch versions a lot)
- Connecting screen spinner (fun to watch)
- Fullbright
- HUD Widgets
  - FPS Counter
  - Potion Statuses

Feel free to suggest more!
